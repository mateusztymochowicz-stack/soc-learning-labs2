# SOC Learning Labs (Bash + Linux)

Repozytorium do nauki roli SOC Analyst.
Mini-laby, notatki, analiza logów i raporty incydentów.

## Cel
- nauczyć się Linux/Bash pod SOC
- analizować logi systemowe
- budować portfolio pod pierwszą pracę w cyberbezpieczeństwie

## Struktura
- notes/ – notatki z nauki dzień po dniu
- labs/ – scenariusze labów
- data/ – próbki logów
- scripts/ – skrypty analityczne
- reports/ – raporty incydentów
- cheat-sheets/ – ściągi komend
